About This Blog

Welcome to my Markdown blog! This is a lightweight, fast, and modern blog engine built with Rust and designed for developers who love writing in Markdown.

Features

  • Markdown-to-HTML conversion with YAML frontmatter
  • Hot reload - changes to content files are automatically detected
  • Fast static file serving
  • Responsive HTML templates
  • Tag support for organizing posts
  • Clean, minimal design focused on readability

Getting Started

To add a new post, simply create a .md file in the content/ directory with YAML frontmatter:

---
title: "My New Post"
date: "2025-09-15"
tags: ["example", "blog"]
---

# Your content here!

Write your blog post in **Markdown** format.

The blog will automatically detect the new file and make it available immediately!
